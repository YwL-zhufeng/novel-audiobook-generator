# Novel Audiobook Generator - Web UI Design

## 目标
创建一个直观、美观的 Gradio Web 界面，让非技术用户也能轻松生成有声书。

## 页面结构

### 1. 顶部导航栏
- Logo + 标题
- 深色/浅色主题切换
- 帮助/文档链接

### 2. 主工作区（标签页布局）

#### 标签页 1: 快速开始
- 文件上传区（拖拽支持）
- TTS 后端选择（ElevenLabs / XTTS / Kokoro）
- API Key 输入（条件显示）
- 生成按钮
- 实时进度条
- 下载结果

#### 标签页 2: 高级配置
- **声音设置**
  - 声音克隆上传（支持多个声音）
  - 声音参数调节（stability, similarity_boost）
  - 声音预览播放
  
- **文本处理**
  - 分块大小调节
  - 对话检测开关
  - 语言选择
  
- **输出设置**
  - 格式选择（MP3 / WAV / M4B）
  - 比特率
  - 音量标准化
  - 分章节输出

#### 标签页 3: 角色配音
- 自动角色检测结果显示
- 角色-声音分配表格
- 旁白声音选择
- 试听每个角色的声音

#### 标签页 4: 批量处理
- 多文件上传
- 队列显示
- 批量配置应用

### 3. 底部状态栏
- 版本信息
- 系统状态（GPU/CPU）
- 日志输出区域（可折叠）

## 交互细节

### 文件上传
- 支持 TXT, EPUB, PDF
- 显示文件信息和预览（前1000字）
- 文件大小限制提示

### 实时反馈
- 进度条带百分比
- 当前步骤文字说明
- 预计剩余时间
- 取消按钮

### 声音克隆
- 上传音频后自动克隆
- 显示克隆进度
- 提供测试文本朗读

### 结果展示
- 生成的音频播放器
- 文件大小、时长信息
- 下载按钮
- 重新生成选项

## 技术实现

### 依赖
```
gradio>=4.0.0
gradio-pdf  # PDF预览支持
```

### 架构
- `webui.py` - 主入口
- `ui/` - UI组件模块
  - `components.py` - 可复用组件
  - `handlers.py` - 事件处理
  - `state.py` - 会话状态管理

### 主题设计
- 使用 Gradio 的自定义 CSS
- 深色模式适配
- 响应式布局（支持移动端）
